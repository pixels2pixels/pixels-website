# Pixels Website

Official website repository for the Pixels application.

## Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/pixels2pixels/pixels-website.git
cd pixels-website
```

## Deployment

This repository is configured for easy deployment. Push to the `main` branch to trigger a deployment.

```bash
git add .
git commit -m "Your commit message"
git push origin main
```
